Introduction to Programming

Description: Programming design involves planning the logic and structure of a program before writing the code to ensure it is efficient, maintainable, and scalable. It can refer to high-level methodologies like top-down design, or low-level, reusable solutions to common problems known as design patterns.

Code: CS101

Credits: 3

Applied Physics

Description: Applied science is the application of existing scientific knowledge and theoretical principles to solve practical problems, develop new technologies, or invent useful products. Unlike pure science (or theoretical science), which aims to understand the fundamental laws of nature for knowledge's sake, applied science is goal-oriented and focused on practical outcomes.

Code: PH103

Credits: 3

Data Structures and Algorithms

Description: Data structures are ways of organizing data, while algorithms are sequences of steps to solve a problem efficiently using those structures. Various image resources are available for visual representations of these concepts.

Code: CS205

Credits: 4

Communication Skills

Code: CM201
